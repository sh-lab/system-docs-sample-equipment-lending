# ユースケース設計書

## ユースケースID
UC-003

## ユースケース名
貸出申請を承認または却下する

## 概要
管理者がマイページの申請一覧から個々の貸出申請を確認し、承認または却下を行う。

## アクター
- 管理者

## 前提条件
- 管理者がログインしている
- 承認待ちの貸出申請が存在する

## 基本フロー
1. 管理者がマイページから申請一覧を表示する
2. 管理者が対象の申請を選択し、申請詳細画面を表示する
3. 管理者が承認または却下を選択し、コメントを入力する
4. 管理者が処理を確定する
5. システムが申請状態を更新する
6. システムが処理完了メッセージを表示する

## 代替フロー
- 管理者が却下を選択した場合、申請は「却下」状態となる

## 事後条件
- 承認時、申請は貸出中状態となり、備品は貸出中状態を維持する
- 却下時、申請は「却下」状態となり、利用者未確認の状態で保持される
