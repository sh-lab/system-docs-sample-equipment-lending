# ユースケース設計書

## ユースケースID
UC-007

## ユースケース名
管理者備品検索を行う

## 概要
管理者が管理者備品検索画面で備品を検索し、管理者備品編集画面へ遷移する。

## アクター
- 管理者

## 前提条件
- 管理者がログインしている

## 基本フロー
1. 管理者がマイページから管理者備品検索画面へ遷移する
2. 管理者が備品名、備品種別、備品状態、システム登録日などの検索条件を入力する
   - 備品種別は、備品種別マスタで管理された候補から選択する
3. システムが検索条件を検証する
4. システムが条件に一致する備品一覧を表示する
5. 管理者が新規登録または対象備品の編集を選択し、管理者備品編集画面へ遷移する

## 代替フロー
- 検索条件に一致する備品が存在しない場合、検索結果が0件である旨を表示する

## 事後条件
- 管理者が検索結果を確認し、管理者備品編集画面で備品登録または備品情報更新へ進める
