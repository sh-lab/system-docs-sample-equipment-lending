# ユースケース設計書

## ユースケースID
UC-006

## ユースケース名
却下された申請を確認する

## 概要
利用者が却下された申請の内容を確認し、申請を完了状態にする。

## アクター
- 利用者

## 前提条件
- 利用者がログインしている
- 申請状態が「却下」であり、利用者未確認の申請が存在する

## 基本フロー
1. 利用者がマイページから却下された申請一覧を表示する
2. 利用者が対象の申請を選択し、申請詳細画面を表示する
3. 利用者が却下内容を確認する
4. 利用者が確認完了操作を行う
5. システムが申請を完了状態に更新する

## 代替フロー
- 対象申請が存在しない場合は処理を行わない

## 事後条件
- 申請は完了状態となり、利用者のマイページ一覧から非表示となる
