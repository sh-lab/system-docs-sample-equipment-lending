# ユースケース設計書

## ユースケースID
UC-004

## ユースケース名
備品の返却を申請する

## 概要
利用者がマイページから貸出中の申請を確認し、返却申請を行う。

## アクター
- 利用者

## 前提条件
- 利用者がログインしている
- 対象備品が当該利用者に貸出中である

## 基本フロー
1. 利用者がマイページから自身の申請一覧を表示する
2. 利用者が対象の申請を選択し、申請詳細画面を表示する
3. 利用者が返却申請を選択し、コメントを入力する
4. システムが申請内容を検証する
5. システムが返却申請を登録する
6. システムが申請完了メッセージを表示する

## 代替フロー
- 貸出中でない申請に対して返却申請を行った場合、エラーメッセージを表示する

## 事後条件
- 返却申請が登録され、申請は返却確認待ち状態となり、備品は貸出中状態を維持する
