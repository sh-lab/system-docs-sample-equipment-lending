# ユースケース設計書

## ユースケースID
UC-002

## ユースケース名
備品の貸出を申請する

## 概要
利用者が備品検索結果から対象備品を選択し、貸出申請を行う。

## アクター
- 利用者

## 前提条件
- 利用者がログインしている
- 対象備品が貸出可能な状態である

## 基本フロー
1. 利用者が備品検索結果から対象備品を1件以上選択する
2. 利用者が貸出申請画面へ遷移する
3. 利用者が申請コメントを入力し、申請を実行する
4. システムが申請内容を検証する
5. システムが貸出申請を登録する
6. システムが申請完了メッセージを表示する

## 代替フロー
- 備品が既に貸出申請中、貸出中、貸出不可、または廃棄の場合、申請エラーメッセージを表示する

## 事後条件
- 貸出申請が登録され、申請は承認待ち状態となり、対象備品は貸出申請中状態となる
