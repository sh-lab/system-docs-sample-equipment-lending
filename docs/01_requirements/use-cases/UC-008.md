# ユースケース設計書

## ユースケースID
UC-008

## ユースケース名
備品を登録する

## 概要
管理者が管理者備品検索画面から管理者備品編集画面へ遷移し、新規備品を登録する。

## アクター
- 管理者

## 前提条件
- 管理者がログインしている

## 基本フロー
1. 管理者が管理者備品検索画面を表示する
2. 管理者が新規登録として管理者備品編集画面を開く
3. 管理者が備品名、備品種別、初期状態などの必要情報を入力する
   - 備品種別は、備品種別マスタで管理された候補から選択する
4. 管理者が登録を確定する
5. システムが入力内容を検証する
6. システムが備品を登録する
7. システムが登録完了メッセージを表示する

## 代替フロー
- 必須項目が不足している場合、システムはエラーメッセージを表示し登録しない
- 備品種別にマスタ未登録の値が指定された場合、システムはエラーメッセージを表示し登録しない

## 事後条件
- 新規備品が登録され、指定した状態およびシステム登録日で貸出管理対象となる
