# ユースケース設計書

## ユースケースID
UC-001

## ユースケース名
備品を検索する

## 概要
利用者がマイページから備品検索画面へ遷移し、貸出対象となる備品を検索する。

## アクター
- 利用者

## 前提条件
- 利用者がログインしている

## 基本フロー
1. 利用者がマイページから備品検索画面へ遷移する
2. 利用者が検索条件を入力する
3. システムが検索条件を検証する
4. システムが条件に一致する備品一覧を表示する

## 代替フロー
- 検索条件に一致する備品が存在しない場合、検索結果が0件である旨を表示する

## 事後条件
- 備品検索結果が表示される
