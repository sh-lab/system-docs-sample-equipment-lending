# ユースケース一覧

| ID     | ユースケース名         | 概要                                           |
|--------|------------------------|------------------------------------------------|
| UC-001 | 備品を検索する         | 利用者が貸出可能な備品を検索する               |
| UC-002 | 備品の貸出を申請する   | 利用者が備品の貸出申請を行う                   |
| UC-003 | 貸出申請を承認または却下する     | 管理者が利用者からの貸出申請を承認または却下する |
| UC-004 | 備品の返却を申請する       | 利用者が貸出中の備品について返却申請を行う                   |
| UC-005 | 備品の返却を確認する       | 管理者が返却申請に基づき実物を確認し返却を確定する           |
| UC-006 | 却下された申請を確認する       | 利用者が却下された申請の内容を確認し、申請を完了状態にする           |
| UC-007 | 管理者備品検索を行う       | 管理者が備品を検索し、登録・状態更新へ遷移する           |
| UC-008 | 備品を登録する       | 管理者が新規備品を登録し、貸出管理対象に追加する           |
| UC-009 | 備品情報を更新する       | 管理者が登録済み備品の備品名、状態、備考を更新し、貸出可否へ反映する           |
